How it began

It all started with Minecraft Redstone. I was nine years old at that time and had no huge understanding about how coding works in general. With Redstone I was able to build and create systems using repeaters, buttons etc. This really helped me understand the core logic behind the programming. Sounds ridicolous I know, but trust me this is how it really began!

Since 2019, after atending at a Junior scripting academy, that curiosity has grown into a full stack of languages and platforms: writing LuaU scripts for Roblox games with millions of visits, building Java plugins for Minecraft servers and so on.
